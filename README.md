## Prerequisites for running (OS X)
##### Java
- Install maven.
```
brew install maven
```

##### C++
- Install gcc and make sure it's on your path variable.
```
brew install gcc
```
- Install boost to /usr/local/include/boost.
```
brew install boost
```
- Install command line tools.
```
xcode-select --install
```
